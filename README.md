# No Training Limit Oblivion

Unlimited training sessions for Oblivion Remastered.

## Getting Started

```bash
git clone --recurse-submodules git@github.com:Zebrina/commonlibob64.git commonlibob64
git clone --recurse-submodules git@github.com:Zebrina/BGSScriptExtenderPluginTools.git BGSScriptExtenderPluginTools
git clone git@github.com:Zebrina/NoTrainingLimitOBSE.git NoTrainingLimit
cd NoTrainingLimit
xmake build
```
